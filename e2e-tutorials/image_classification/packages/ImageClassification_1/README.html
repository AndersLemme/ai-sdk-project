Pipeline ImageClassification (None)

This pipeline runs a TensorFlow Lite model on an Industrial Edge device. The model was trained to recognize and classify images of following Siemens SIMATIC automation products: ET 200AL, ET 200eco PN, ET 200SP, S7-1200, S7-1500. The pipeline is designed to be fed from Vision Connector via ZMQ in BayerRG8 pixel format. The pipeline output is to be sent to the Databus.

Inputs
  • vision_payload (ImageSet)
Outputs
  • prediction (String)

PythonComponent inference

This component uses a trained TensorFlow Lite image classification model that reads the vision_payload input and produces a prediction as an output.
